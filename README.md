# CollatzVisualisation
We implement a MATLAB animated visualisation of the generalised Collatz system: 

```math
C(n)=\begin{cases}\frac{n}{2} & \text{if }n\equiv 0\pmod{2}\\3n+1 & \text{if }n\equiv1\pmod{2}\end{cases}
```
